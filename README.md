### About

Go (golang) library which provides extended testing helpers and utilities.

Created by [Jay Taylor](http://jaytaylor.com/) and used by [Gigawatt](http://gigawatt.io/).

### Requirements

* Go version 1.3 or newer

### Running the test suite

    go test ./...
